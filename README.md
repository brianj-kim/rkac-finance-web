This web app - RKAC Finance is developed for my church - Regina Korean Alliance Church to help the treasurer in recording the income (offering, gift, etc.) and generate donation receipt for the tax report (Canada Revenu Agency, CRA). This is a MVP for the project and will be improved as time goes by.

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000/income](http://localhost:3000/income) with your browser to see the result.

## PostgreSQL Database

This project requires to use PostgreSQL Database server via docker (v. 16 or later)
